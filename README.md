### Данный репозиторий является выполнением тз
Для удобства файл разделен на различные папки:
1. в папке "tests" находится необходимые компоненты для теста
2. в папке "API" находится тест 
3. в папке "main_log" находятся основная логика теста
4. в файле "check" находятся необходимые проверки для теста
5. в файле "func" находятся основные функции для теста

Для установки всех зависимостей присутствует "requirements" файл
Для отчетности присутствует allure репорт
